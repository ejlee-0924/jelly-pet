# Jelly Pet

Virtual pet companion for Claude Code that grows with your coding sessions.

Code more, feed jelly, evolve your pet.

## How It Works

```
You code with Claude Code
  → Output tokens are counted
  → Tokens convert to Jelly (500 tokens = 1 jelly)
  → Your pet eats jelly and grows
  → Evolve through 5 stages!
```

## Evolution Stages

| Stage | Jelly | Time (~casual user) |
|:-----:|------:|:-------------------:|
| Egg | 0 | Start |
| Baby | 100 | ~1 week |
| Pro | 500 | ~1 month |
| Master | 2,000 | ~3-4 months |
| Legend | 10,000 | ~1 year+ |

## Pet Species

Choose from 5 pets, each with unique ASCII art evolution:

- Cat
- Dog
- Hamster
- Rabbit
- Hedgehog

Switch anytime — your jelly and level stay.

## What You See

**Statusline (always visible):**
```
  /\ /\        Opus 4.6  ████████░░░░ 62%
 ( o.o )       Jelly 42  Lv.1 Egg
  > ^ <        Streak 3d  Evolve ██░░░░ 42%
  '---'
```

**Session start:** Your pet greets you with personality-driven messages.

**Session end:** Report showing jelly earned, streak, and evolution progress.

## Commands

| Command | Description |
|---------|-------------|
| `/jelly-pet:status` | Current pet, jelly, level, progress |
| `/jelly-pet:switch` | Change pet species |
| `/jelly-pet:evolve` | View evolution tree |

## Streak Bonus

Code every day for bonus jelly:

| Streak | Multiplier |
|--------|:----------:|
| 1-3 days | x1.0 |
| 4-7 days | x1.5 |
| 8+ days | x2.0 (MAX) |

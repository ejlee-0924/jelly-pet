이게 뭐야?

코딩할수록 자라나는 내 펫

Jelly Pet은 Claude Code에서 사용하는 플러그인이에요.
Claude Code로 코딩하면 사용한 만큼 젤리가 쌓이고,
내 펫이 점점 자라서 진화해요!

Claude Code가 뭐예요?
Anthropic이 만든 AI 코딩 도구예요. 터미널(까만 화면)에서 AI와 대화하면서 코드를 작성하거나 업무를 처리할 수 있어요. 개발자가 아니어도 쓸 수 있습니다!

설치 방법

2가지 방법이 있어요

가장 쉬운 방법! 명령어 하나면 설치 완료.
  1. 마켓플레이스 등록 Claude Code를 열고 아래 명령어를 입력하세요:
    /plugin marketplace add https://github.com/ejlee-0924/jelly-pet

    이 명령은 한 번만 하면 돼요. jelly-pet을 찾을 수 있는 장소를 등록하는 거예요.
    마켓플레이스가 뭐예요? 플러그인을 모아놓은 "앱스토어" 같은 거예요. 한 번 등록하면 그 안의 플러그인을 자유롭게 설치할 수 있어요.
  2. 플러그인 설치 마켓플레이스 등록 후 설치 명령어를 입력하세요:
    /plugin install jelly-pet

    자동으로 다운로드 + hooks 연결 + 스킬 등록까지 한 번에!
  3. 펫 선택하고 이름 짓기 Claude Code를 재시작하면 환영 메시지가 나와요!
    /jelly-pet:switch 명령으로 펫을 고르고 이름을 지어주세요.

    선택 가능: 🐱고양이 🐶강아지 🐹햄스터 🐰토끼 🦔고슴도치
플러그인 설치가 안 될 때 또는 직접 코드를 보면서 설치하고 싶을 때 이 방법을 쓰세요.
  1. 프로젝트 다운로드 Claude Code 터미널에서 아래 명령어를 입력하세요:
    ! git clone https://github.com/ejlee-0924/jelly-pet.git ~/.jelly-pet-plugin
    ! 가 뭐예요? Claude Code에서 터미널 명령어를 직접 실행하는 방법이에요. 앞에 ! 를 붙이면 됩니다.
  2. hooks 연결하기 Claude Code 설정 파일에 jelly-pet hooks를 추가해야 해요.
    가장 쉬운 방법: Claude Code에서 이렇게 말하세요:

    jelly-pet hooks를 settings.json에 연결해줘

    AI가 알아서 설정 파일을 수정해줘요!

    직접 하고 싶다면 ! open ~/.claude/settings.json 으로 열고
    hooks에 아래를 추가하세요:
    // SessionStart에 추가
    "command": "node ~/.jelly-pet-plugin/hooks/session-start.js"

    // SessionEnd에 추가
    "command": "node ~/.jelly-pet-plugin/hooks/session-end.js"

    // statusLine 변경
    "command": "node ~/.jelly-pet-plugin/lib/statusline.js"
  3. 펫 선택하고 이름 짓기 Claude Code를 재시작하면 환영 메시지가 나와요!
    /jelly-pet:switch 명령으로 펫을 고르고 이름을 지어주세요.

    선택 가능: 🐱고양이 🐶강아지 🐹햄스터 🐰토끼 🦔고슴도치
명령어

사용할 수 있는 명령어

/jelly-pet:status 현재 펫 상태 보기
/jelly-pet:switch 다른 펫으로 변경 + 이름 설정
/jelly-pet:rename 펫 이름 바꾸기
/jelly-pet:evolve 진화 트리 + 진행 상황 보기
자주 묻는 질문

궁금한 거 있죠?

Q. 펫을 바꾸면 젤리 사라져요?
아뇨! 젤리, 레벨, 진화 단계 전부 유지됩니다. 겉모습(ASCII 아트)만 바뀌어요.
Q. 하루에 여러 세션 하면 젤리 더 많이 모여요?
네! 세션마다 젤리가 쌓여요. 많이 쓸수록 빠르게 진화합니다.
Q. 며칠 안 하면 펫이 죽어요?
죽지 않아요! 다만 연속 접속 보너스(스트릭)가 1일로 리셋됩니다.
Q. 개발자가 아닌데 써도 돼요?
물론이요! Claude Code 자체가 비개발자도 쓸 수 있어요. 코딩 외에도 문서 작성, 기획 등 다양한 작업에 사용하면 젤리가 쌓입니다.
Q. 데이터가 어딘가로 전송되나요?
아뇨. 모든 데이터는 내 컴퓨터(~/.jelly-pet/)에만 저장돼요. 외부 서버 연결 없습니다.
